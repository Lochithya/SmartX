# SmartX – Smart Home Application UI/UX Wireframe Design

## Introduction

SmartX is a smart home management application wireframe project developed as part of the **SENG 21263** module. The primary objective of this project was to design a modern, user-friendly, and responsive smart home application interface that allows users to manage and control their smart devices from a single platform.  

The wireframes for this application were designed using **Figma**, focusing on clean UI structure, logical user flows, accessibility, and ease of navigation. The application was created with the intention of simplifying smart home management by replacing the need for multiple remote controls and providing centralized control through one application.  

This project also emphasized practical UI/UX principles such as responsive interactions, minimal complexity, user confirmations for sensitive actions, and realistic workflow designs to improve the overall user experience.

# Project Objective

The purpose of **SmartX** is to provide users with a centralized smart home control system where they can manage connected devices, rooms, and smart features through a single mobile application.  

Instead of using multiple remote controls or separate applications for different devices, SmartX enables users to access and control all smart home functionalities in one place, improving convenience, accessibility, and efficiency.
